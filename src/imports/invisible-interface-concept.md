# 🟡 1. ENTRY EXPERIENCE — ONE DOT

The entire screen is **pure off-white**.

Nothing else.

In the center there is **one tiny black dot**.

When the cursor approaches:

The dot **gently expands**.

Text fades in beside it:

```
Husain Bhatiya
UI / UX Designer
```

The dot becomes a **navigation nucleus**.

From this one point the entire site unfolds.

Very unexpected.

Very minimal.

---

# 🟡 2. HERO SECTION — FLOATING COMPOSITION

Instead of a layout grid:

Everything floats in **perfect balance**.

Elements:

• Your portrait
• Your name
• One sentence
• Two buttons

But positioned asymmetrically like **editorial art direction**.

Example layout:

Left side → your portrait
Right side → large typography

But the spacing is **very airy**.

Huge breathing room.

---

### Animation

Nothing pops.

Everything **glides slowly**.

Subtle motion:

• portrait floats 2px
• text letters slightly shift spacing
• background grain slowly moves

It feels alive.

---

# 🟡 3. YOUR IMAGE — CREATIVE USE

Your portrait is not just placed.

It becomes an **interactive art object**.

When user moves mouse:

The image tilts slightly in 3D.

A **soft light reflection** moves across your glasses.

When user clicks the image:

It splits into **three vertical slices**.

Each slice reveals information:

Left slice → Designer
Center slice → Thinker
Right slice → Tester

This is extremely minimal but very memorable.

---

# 🟡 4. PROJECT SECTION — SLIDING PANELS

No grids.

Projects appear as **large horizontal panels**.

Each panel almost fills the screen.

Layout:

```
[ Huge Project Title ]

Minimal description

[ floating device mockup ]
```

When user scrolls:

Panels slide **sideways**, not vertically.

The experience feels like **moving through an exhibition**.

---

### Panel animation

Panels expand slightly on hover.

Background color softly shifts to match the project.

Example:

Food It → warm orange tone
Lazy Project → neutral beige
Testing → dark graphite

Everything transitions **very slowly and elegantly**.
